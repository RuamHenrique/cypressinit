# 🧪 Projeto de Automação de Testes - Cypress + Cucumber

## 📌 Descrição

Projeto de automação de testes utilizando Cypress com Cucumber (BDD), aplicando Gherkin para escrita de cenários.

O sistema testado é o Portal da Transparência do Governo Federal.

---

## 🚀 Tecnologias

- Cypress
- Cucumber (BDD)
- Gherkin
- JavaScript

---

## 📁 Estrutura

cypress/
 ├── e2e/
 │   ├── features/
 │   ├── steps/
 ├── support/
 ├── fixtures/

---

## 🎯 Objetivo

Validar funcionalidades básicas do portal:

- Acesso à página
- Pesquisa de dados
- Validação de URL
- Elementos da interface
- Cenários com Scenario Outline

---

## 🧪 Execução

npm install  
npx cypress open

---

## ✔️ Resultado

12 testes automatizados executados com sucesso.
